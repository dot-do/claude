## Quick Example

```typescript title="client.ts"
import { createClaudeClient } from '@dotdo/claude'

const claude = createClaudeClient({
  url: 'wss://your-worker.workers.dev/rpc',
  callbacks: {
    onMessage: (msg) => console.log('Claude:', msg),
    onTodoUpdate: (update) => console.log('Todos:', update.todos),
  }
})

const session = await claude.createSession({ cwd: '/workspace' })
await claude.sendMessage(session.id, 'Build a todo app in React')
```

## Features

| Feature | Description |
|---------|-------------|
| **Session Management** | Create, resume, and manage Claude Code sessions |
| **Real-time Streaming** | WebSocket and SSE support for live updates |
| **Event Callbacks** | Typed callbacks for todos, plans, tools, and results |
| **Edge Deployment** | Built for Cloudflare Workers and Durable Objects |
| **React Components** | Production-ready terminal, chat, and IDE components |
| **Type Safety** | Full TypeScript support with comprehensive types |

## Architecture

```
┌─────────────────┐     WebSocket/RPC     ┌──────────────────┐
│  Your Web App   │ ◄──────────────────► │  Cloudflare DO   │
│  (ClaudeClient) │                       │  (ClaudeCode)    │
└─────────────────┘                       └────────┬─────────┘
                                                   │
                                                   │ stdin/stdout
                                                   ▼
                                          ┌──────────────────┐
                                          │   Claude Code    │
                                          │      CLI         │
                                          └──────────────────┘
```
