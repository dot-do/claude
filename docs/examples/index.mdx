## Getting Started

Each example is self-contained and demonstrates specific features of the SDK:

- **Basic Chat** - Start here to understand the fundamentals of sending messages and receiving responses
- **IDE Layout** - Learn how to build complex, multi-panel interfaces with Claude Code integration
- **Embedded Terminal** - See how to embed the full terminal experience within your own UI

Clone the repository and run any example locally to experiment with the code.
