The Terminal package provides a comprehensive set of React components for building interactive terminal interfaces and chat experiences.

## Available Components

<Cards>
  <Card title="Terminal" href="/terminal/components/terminal">
    The main terminal emulator component with full xterm.js integration, keyboard handling, and ANSI rendering.
  </Card>
  <Card title="Chat" href="/terminal/components/chat">
    Conversational interface component for displaying message threads between users and Claude.
  </Card>
  <Card title="FileTree" href="/terminal/components/filetree">
    Hierarchical file browser component with expand/collapse, icons, and selection support.
  </Card>
  <Card title="Layout" href="/terminal/components/layout">
    Flexible layout primitives for arranging terminal panels, sidebars, and content areas.
  </Card>
  <Card title="DiffViewer" href="/terminal/components/diff-viewer">
    Side-by-side and unified diff display component for visualizing code changes.
  </Card>
  <Card title="SessionPanel" href="/terminal/components/session-panel">
    Session management panel for viewing, switching, and controlling active Claude sessions.
  </Card>
  <Card title="ErrorBoundary" href="/terminal/components/error-boundary">
    Error boundary component for graceful error handling and recovery in the UI.
  </Card>
</Cards>

## Quick Overview

| Component | Purpose |
|-----------|---------|
| `Terminal` | Full-featured terminal emulator |
| `Chat` | Message-based conversation UI |
| `FileTree` | File system navigation |
| `Layout` | Page structure and panels |
| `DiffViewer` | Code diff visualization |
| `SessionPanel` | Session management |
| `ErrorBoundary` | Error handling wrapper |
